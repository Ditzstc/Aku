## PENJELASAN: Cara Create File JSON & Get API Token GitHub

1. **Buat repositori** (nama bebas).

2. **Buat file JSON berikut:**
   - `Activity.json`
   - `sender.json`
   - `db.json`

3. Buka **Profile â†’ Settings â†’ Developer Settings**.

4. Pilih **Personal Access Tokens â†’ Fine-grained Tokens**.

5. Klik **Generate New Token**.

6. Isi **Token Name** sesuai keinginan.

7. Pilih **Expiration (90 Days)**.

8. Pada **Repository Access**, pilih **Only select repositories**, lalu pilih repo yang lu buat tadi.

9. Klik **Add Permissions**, cari **Contents**, centang, lalu ubah izin jadi **Read and Write**.

10. Klik **Generate Token**, lalu **salin token** lu.

11. Tempelkan token GitHub di file `script.js`.
